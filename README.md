# Tetris (Mobile Web)

A lightweight, mobile-friendly Tetris game (static HTML/CSS/JS).

## Play locally

From this folder:

```bash
python -m http.server 8008
```

Then open:

- Desktop: `http://127.0.0.1:8008/`
- Phone/iPad (same Wi‑Fi): `http://<your-computer-ip>:8008/`

## Mobile tips

- iPhone/iPad Safari: Share → **Add to Home Screen** (plays like an app).

## Controls

- Touch: swipe left/right/down on the board, tap to rotate.
- Buttons: left / rotate / right / down.
- Keyboard (desktop):
  - Move: Arrow keys (← → ↓)
  - Rotate: Arrow Up (↑)
  - Pause: `P`

## Hosting

This project is static and can be hosted on GitHub Pages, Netlify, or Cloudflare Pages.
